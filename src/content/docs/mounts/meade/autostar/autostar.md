## Features

Do not use this driver with LX200 Classic telescope as it uses a seperate driver. Even though you can use this driver to control LX200 GPS or LX200 16" telescopes, you won't be able to use the extra features provided by these devices

## Operation

In addition to the regular telescope functions, the lx200autostar driver provides users with information on the current firmware
