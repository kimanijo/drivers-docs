## Features

The lx200classic supports functions that were removed with the introduction of Autostar:

-   Ability to read object name from controller.
-   Setting Max Slew Rate.
-   Setting Maximum and Minimum Slew Limtits.
-   Support for all library commands (lookups for objects in NGC, IC, UGC, Caldwell, Arp, Abell, Messier).

## Operation

Do not use this driver with new Autostar and GPS based Meade. It supports functions that no longer exists in Autostar
